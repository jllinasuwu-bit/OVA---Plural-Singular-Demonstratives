Patrón unoSingular → Plural: Al haber más de un objeto, se añade -s al sustantivo (o -es si termina en -ch, -sh, -x, -s). El verbo cambia de is a are.

Patrón dosThis vs. That: This se usa para un objeto singular cerca del hablante; that para un objeto singular lejos.

Patrón tresThese vs. Those: These es el plural de this (cerca, varios); those es el plural de that (lejos, varios).

📌 a vs an — ¿cuándo usar cada uno?

Generalmente se usa an cuando el sustantivo que viene después empieza con vocal (a, e, i, o, u). En todos los demás casos se usa a.

Usa «a»
This is a desk
This is a pen
This is a chair
Usa «an»
This is an eraser
⟶ empieza con e (vocal)
⚠ Excepción importante: Algunas palabras empiezan con la letra u pero suenan como si empezaran con «y» (en español). En esos casos se usa a, no an.
Ejemplos: a uniform, a university — la u suena como «yu», no como vocal pura. En este nivel, el único objeto que requiere an es eraser. ¡Tenlo presente!
⚠ Error frecuente — Interferencia del español

En español el pronombre sujeto puede omitirse. En inglés es obligatorio.

✗ "Is a pen."

✓ "This is a pen."  /  ✓ "That is a pen."

Todo enunciado en inglés debe tener un sujeto explícito. Nunca se omite.

⚠ Error frecuente — These / Those con verbo is

✗ "These is chairs."   ✗ "Those is desks."

✓ "These are chairs."   ✓ "Those are desks."

These y those son siempre plurales → siempre usan are.

⚠ Error frecuente — a antes de sonido vocálico

✗ "This is a eraser."

✓ "This is an eraser."

Cuando el sustantivo empieza con vocal (a, e, i, o, u), el artículo cambia a an. La primera letra del sustantivo es lo que determina si se usa a o an.
